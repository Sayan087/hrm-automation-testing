# 🧪 OrangeHRM Automation Testing Framework

## 📌 Project Overview

This project is an **Automation Testing Framework** built to test the **OrangeHRM Demo Application** using **Java, Selenium WebDriver, and Test Automation best practices**.

The framework is designed following industry standards to ensure scalability, maintainability, and reusable test components.

---

## 🎯 Objectives

* Automate critical HRM workflows
* Validate core functionalities of OrangeHRM
* Reduce manual testing effort
* Ensure application stability through automated regression testing

---

## 🧰 Tech Stack

* **Language:** Java
* **Automation Tool:** Selenium WebDriver
* **Build Tool:** Maven
* **Test Framework:** TestNG / JUnit
* **Design Pattern:** Page Object Model (POM)
* **Version Control:** Git & GitHub

---

## 🏗️ Framework Design

This project follows:

✔ Page Object Model (POM)
✔ Reusable utilities & helpers
✔ Separation of test logic and page actions
✔ Clean folder structure for scalability

## ▶️ How to Run Tests

```

### 2️⃣ Open in IntelliJ IDEA

### 3️⃣ Install Dependencies

Maven will download required dependencies automatically.

### 4️⃣ Run Tests

Run test classes from:

```
